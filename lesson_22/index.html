Задача №1

                            let someVar = 0;
                            ++someVar;
                            if (someVar) {
                            console.log(someVar);
                            }
Відповідь: 1. Інкремент ++ збільшує значення на одиницю.

Задача №3

Питання: Що потрапить в консоль? 
                                if (2 * 20 <= 10 || 30 / 2 < 5 && 10 <= "10" || 20 === "20") {
                                    console.log('000');
                                }
Відповідь: Нічого. Спочатку виконаються аріфметичны дії, потім логічні. Так як && має перевагу над ||, то перевірятимуться умови 30 / 2 < 5 - false та 10 <="10" - true тому, що обидва операнди приводяться до числа і ця умова виконана. Отже false || true в підсумку маємо false.

Задача №4

Завдання: Створіть функцію, яка повертає результат ділення числа a на число b з додаванням рядка "Результат ділення: " Викличте функцію передаючі різні значення, у тому числі не передаючи зовсім. Функція не має повертати NaN, Infinite або помилку
Відповідь: 
                                function getQuotient(a, b) {
                                    const result = a / b
                                    if (isNaN(result) || result === Infinity)
                                        alarm (`Введіть коректні значення`)
                                }
                                return `Результат ділення: ${result}`

                                console.log(getQuotient(8, 4))
                                console.log(getQuotient(8, 0))
                                console.log(getQuotient())
                                console.log(getQuotient(5, `John`))
                                console.log(getQuotient(-12, 3))
